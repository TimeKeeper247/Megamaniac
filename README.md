# MegaManiac

by TimeKeeper247 (N9452257)

"Galaga" clone console program

### Controls:

- **a** --- Move left
- **d** --- Move Right
- **s** --- Shoot
- **l** --- Cycle levels
- **r** --- Restart
- **q** --- Quit
### (Only on level 5)
- **z** --- veer bullet to left
- **c** --- veer bullet to right
